# 📖 Project Guide: Hotel Booking Demand Analysis

## 🛠️ Tools Used
* **Google Sheets:** Data Cleaning & Initial EDA
* **BigQuery SQL:** Advanced Data Manipulation
* **Tableau/Sheets:** Data Visualization
* **GitHub:** Project Documentation

## 📂 Folder Structure
* `scripts/`: Contains SQL queries used for data processing.
* `visualizations/`: High-resolution images of the final dashboard.
* `data/`: (Optional) Link to the original dataset source.

## 🚀 How to Replicate
1. **Clean:** Run the `scripts/data_cleaning.sql` on the raw dataset.
2. **Process:** Use the VLOOKUP methodology detailed in the main README.
3. **Visualize:** Refer to the `visualizations/` folder to replicate the chart styling.
