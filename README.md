# Hotel Booking Demand Analysis
## 1. Introduction
This project analyzes hotel booking data to understand factors influencing cancellations, booking trends, and revenue metrics. This analysis serves as a comprehensive case study for data-driven decision-making in the hospitality industry.

## 2. Business Problem
- **Objective:** To identify key drivers of booking cancellations and optimize occupancy strategies.
- **Key Questions:** - What is the cancellation rate?
    - How does lead time impact cancellations?
    - Which month has the highest occupancy?

## 3. Data Source
- [Hotel Booking Demand Dataset (Kaggle)](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand)

## 4. Tools Used
- **Google Sheets:** Data Cleaning & `VLOOKUP`
- **SQL (BigQuery):** Data Manipulation & Analysis
- **GitHub:** Project Documentation
